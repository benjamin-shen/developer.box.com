# リクエストのオプション

Box Sign APIには、署名リクエストを送信する際のカスタマイズと構成のさまざまなオプションが用意されています。開発者はこれらのオプションを使用することで、アプリケーションに固有の要件に合わせてユーザーエクスペリエンスとワークフローをカスタマイズできます。
